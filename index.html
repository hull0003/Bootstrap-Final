CSS:
CSS stands for Cascading Style Sheets, and is a language used to describe how HTML elements are to be displayed. CSS allows for the separation of presentation from content. Multiple HTML pages may share a CSS file, which reduces repetition and allows for reusability, important concepts in programming philosophy. CSS specifications are maintained by the World Wide Web Consortium (W3C).
Github:
Github is a web based service for using Git version control software. It is a platform that allows developers to easily collaborate and work on projects at the same time, with a more user friendly interface than using the command line. Learning how to use it might cause you to develop a nervous tic.
HTML5:
HTML stands for Hypertext Markup Language. It is the standard language for creating web pages and applications. HTML% is the latest version, with several new semantic elements that allow for increased accessibility. HTML specifications are maintained by the World Wide Web Consortium (W3C).
JavaScript:
JavaScript is a high level programming language used along with HTML and CSS to create interactive web pages and applications. It is used to create, control and validate dynamic web content. It can be implemented both client-side and server-side.
PHP:
PHP: Hypertext Preprocessor is an open source server-side scripting language used for web developement that can be embedded into HTML. PHP is an alternative to Microsoft's Active Server Pages (ASP) technology. It is often used in conjunction with a MySQL database. It only made me cry a little bit this semester.
Sass:
Sass is an acronymn for "Syntactically Awesome Style Sheets". It is a preprocessor scripting language that is interpreted or compiled into CSS. The Sass website touts it as "CSS with superpowers". It serves as an extension of CSS that allows for the use of variables, nested rules, inline imports, as well as othe features.
